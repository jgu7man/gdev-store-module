Importar productos Sube un archivo con la información de tus productos. Es necesario que sigas las instrucciones de manera correcta para que sea efectivo.

Instrucciones

  1. El archivo que debes subir, debe ser formato CSV. Es un archivo similar a un tipo de archivo "Excel", incluso es leído por este mismo programa.
  2. Si tienes tu lista de productos en un archivo de Excel, puedes guardar este mismo archivo como CSV al seleccionar, sólo asegúrate de que tu archivo tiene el formato correcto.
    • Sólo el nombre de las columnas en la primera fila
    • Sin encabezados o títulos
    • Sin celdas o filas vacías.
    • Sin celdas combinadas
    • Sin imágenes, gráficos o multimedia
  3. Las columnas que puede o debe tener tu archivo son las siguientes sin espacios ni caracteres especiales y respentando las mayúsculas y minúsculas que se muestran:
    • referencia: El nombre o identificador principal de tu producto,
    • precio: El precio de tu producto, sin signos ni letras. Puede contener decimales,
    • descripcion: Un texto explícito de lo que es tu producto,
    • onStock: (Opcional). El estado de tu producto. Usa sólo las etiquetas true para definir si tienes stock o false de lo contrario,
    • stockCant: (Opcional) Cantidad en stock, si no lo agregas. Tu producto siempre estará disponible para venta,
    • imagenUrl: (Opcional) Una URL de la imagen principal de tu producto, que pueda estar alojada en algún servicio de almacenamiento, o de alguna otro sitio web,
    • SKU: (Opcional) Código identificador en tu tienda
  4. Para guardar tu archivo de Excel como CSV ve a Menú Archivo => Guardar como... => Formato => CSV
  5. Si usas Google Sheets ve a Menú Archivo => Descargar => Valores separados por comas (.csv, hoja actual)
  6. En esta página presiona el botón Importar CSV y arrastra el archivo al cuadro punteado o tócalo para buscar una ubicación en tu dispositivo.
